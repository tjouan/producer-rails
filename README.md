producer-rails
==============

  Ruby on Rails specific macros and tests for [producer][].

[![Version][badge-version-img]][badge-version-uri]


Usage
-----

```ruby
require 'producer/rails'

set :repository,  'git.example:repository_path'
set :app_path,    'deployment_path'

deploy
```
